Oranda

HTML to PDF web application powered by Sinatra using WKHTMLTOpdf

View project on GitHub

Oranda is a simple web application for converting HTML pages to PDF. It provides a basic web interface and a JSON API.

Requirements:

  • Sinatra
  • WKHTMLTOpdf

Usage:

curl curl -F 'url=http://www.joecode.com' http://localhost:4567/pdf

Results:

json {"status":"success", "pdf":"/pdf/71fc24f5-6e0b-4983-bd86-269d78fbdb72.pdf"}
